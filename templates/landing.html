Personal Agents MCP Server

A production-grade Model Context Protocol (MCP) implementation bridge for AgentMail and HubSpot CRM.

The Problem

I was having to connect numerous services via API keys for each AI agent I created.

The AI Solution

This server allows LLMs and AI agents to access the services I need to connect without me having to connect them via API key for each new agent.

Core Capabilities

TOOL

Unified Inbox Intelligence

Aggregrates email signals from services like LinkedIn, Google Alerts, etc. sent to Agentmail. Sends updates on activities via Agentmail.

TOOL

Automated HubSpot Sync

Instantly creates contacts and leads from parsed emails based on defined criteria, email sentiment and intent detection.

Security & Governance

Security Overview: This server utilizes Bearer Token Authentication and Server-Side Secret Management to protect CRM data. By acting as a secure proxy, the system ensures that AI agents interact with HubSpot and AgentMail through a controlled, encrypted gateway where API keys remain 100% private.

● Audit Log: All tool calls are logged with unique Request IDs for traceability and governance.
